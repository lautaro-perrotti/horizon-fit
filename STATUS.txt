❓ TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"Connection refused" en 8088?
  → Docker puede tardar. Espera 30+ segundos
  → Ver logs: docker-compose logs wordpress

"Port already in use"?
  → Windows: netstat -ano | findstr :8088
  →         taskkill /PID <PID> /F
  → Mac/Linux: lsof -i :8088 → kill -9 <PID>

¿Borré algo importante?
  → No hay problema. Docker guarda todo en volúmenes
  → docker-compose down -v && docker-compose up -d
  → Empieza desde cero
